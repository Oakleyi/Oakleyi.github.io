HTML入门教程

在网页中HTML专门负责网页的结构

HTML编辑器

nodepad++，推荐使用vscode。
将vscode翻译成中文界面，下载插件Chinese

HTML基础结构

<!doctype&nbsp;html>文档声明
 <html>
    <head>帮助浏览器或者搜索引擎解析网页
        <meta>
        <title>网页的标题</title>
    </head>
    <body>
        网页显示内容
    </body>
 </html>

以上是网页的基本结构

HTML 在<head>内的标签

  • <meta>用来设置网页的元数据

    • charset 指定网页的字符

    • name 指定的数据的名称

      <keywords 表示网页的关键字>
      <description 用于网站的指定描述>
      具体详情查阅HTML手册

    • content 指定的数据内容

  • <title>网页的标题</title>显示在浏览器的搜索栏，根据title判断网页内容

    HTML 在<body>内的标签

    <body>表示网页的主体，网页所有可见的内容都应该写在body

  • 自结束标签
           像<img>，<imput>

  • 注释：<!– 注释内容 –>

  • 标签属性：在标签中（开始标签或自结束标签）可以设置属性
    属性是一个名值对（x=”y”或’y’）<font  x=”y”>需要改变属性的内容</font>  x是名，y是值

  • 实体语法 <在网页中编写特殊符时会出现问题>

    实体的名字;

  • 语义化标签(1)：<一般来说在块元素里面放行内元素>

    • 块元素：对网页进行布局，在页面中独占一行的标签
      <p>标签在页面中表示一个段落，p标签中不能放任何块元素
      <hgroup>标签为标题组，可以存放一组相关的标题

    • 行内元素：用来包裹文字在页面中不会独占一行
      <em>标签表示语音语调的加重
      <strong>表示强调重要内容

  • 语义化标签（2）：结构化语义标签
    <header>表示网页的头部，可以有多个
    <main>表示网页的主体，只能有一个
    <foot>表示网页的底部
    <nav>表示网页的导航
    <aside>和主题相关的内容（侧边栏）
    <artice>表示一个独立文章
    <section>表示一个独立的区块

    **div 没有语义，就表示一个区块，目前主要的布局元素
span没有任何语义，一般用来选中文字**

  • 语义化标签（3）：列表（list）列表之间可以相互嵌套

    • 有序列表

      <ui></ui>创建

    • 无序列表

      <oi></oi>创建

    • 定义列表

      <dl></dl>创建
      <dt></dt>定义到内容
      <dd></dd>对内容进行解释说明

    • 超链接：从当前页面跳转到其他页面或者当前页面的其他位置,在标签中可以除它外的人和元素

          <a></a>定义超链接
          href属性 指定跳转路径，相对路径绝对路径
              将href属性设置为#回到网页顶部，底部的设计利用id属性
          id属性 元素的唯一标识，在同一个不能重复出现id属性
          target属性，用来指定超链接打开的位置
            可选值：
              _self 默认值在当前页面打开超链接。
              _blank 在一个新的页面打开超链接。

    • 图片标签：用于向当前页面引入一个外部图片
      img属性替换元素，具有块元素和行内元素的特点

      • 使用img标签引入，img是自结束标签
        属性    src属性指定外部图片路径
                   alt属性图片描述搜索引擎根据alt内容识别图片
                   width,height属性
      • 图片的格式

  • 内联框架：用于向当前界面引入一个其他页面

    • 使用<iframe></iframe>标签
                  src属性指定引入网页属性
                 frameborder属性指定内联框架的边框

  • 音视频：

    • audio标签引入外部音频
                 controls属性是否允许用户控制播放
                 aotoplayer属性是否允许自动播放，大部分浏览器都不会自动对音乐进行播放
