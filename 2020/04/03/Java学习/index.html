面向对象 上

面向对象概述

  • 面向对象的特点
    • 封装：面向对象的核心思想，将对象的属性和行为封装起来，不需要外界知道具体实现细节
    • 继承：主要描述类与类之间的关系，通过继承，在无需重新编写原有类的情况下，对原有类的功能进行扩展
    • 多态：发现在继承之后，引用相同类型的变量调用同一个方法所呈现的多种不同行为特征

Java 中的类与对象

  • 类与对象的关系

    • 类是对某一类事物的抽象描述，而对象用于表示现实中该类实物的个体
    • 类用于描述多个对象的共同特征，它是对象的模板，而对象用于描述现实中的个体，它是类的实例
